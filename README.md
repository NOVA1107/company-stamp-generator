# 🏢 公司印章生成器

在线印章生成与文档合成工具。

## 功能

- ✅ 印章生成：圆形、方形、椭圆、五角星四种样式
- ✅ 自定义颜色、尺寸、边框粗细
- ✅ 副标题/编号支持
- ✅ 文档合成：支持 PDF、JPG、PNG 上传
- ✅ 印章拖拽、缩放、旋转、透明度调整
- ✅ PNG 下载和 PDF 导出

## 快速部署

[![Deploy to Vercel](https://vercel.com/button)](https://vercel.com/import/project?template=https://github.com/NOVA1107/company-stamp-generator)

点击上方按钮，一键部署到 Vercel！

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 技术栈

- Next.js 16
- Tailwind CSS
- TypeScript
- jsPDF
- pdfjs-dist
