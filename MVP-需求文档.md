# 公司印章生成器 - MVP 需求文档

**版本**: V1.0  
**日期**: 2026-03-22  
**状态**: 初稿

---

## 1. 产品概述

### 1.1 产品定位
在线印章生成与文档合成工具，用户可通过网页快速生成公司印章，并将印章拖拽至上传的文档中，最终导出带印章的 PDF 文件。

### 1.2 目标用户
- 企业行政人员
- 财务人员
- 创业者 / 小微企业主
- 需要快速制作印章文档的个人

### 1.3 产品价值
- **快捷**: 30秒内完成印章生成
- **专业**: 多种样式可选，满足不同场景
- **便捷**: 印章与文档合成一体化，无需 PS/截图拼接

## 2. 功能需求

### 2.1 核心功能 (MVP)

| 功能模块 | 功能点 | 优先级 | 描述 |
|---------|--------|-------|------|
| 印章生成 | 输入公司名称 | P0 | 文本输入，实时预览 |
| 印章生成 | 4种印章样式 | P0 | 圆形、方形、椭圆、五角星 |
| 印章生成 | 自定义颜色 | P1 | 边框颜色、文字颜色 |
| 印章生成 | 尺寸调整 | P1 | 印章大小、边框粗细 |
| 印章生成 | 副标题/编号 | P2 | 支持添加"合同专用章"、注册号等 |
| 文档合成 | 文件上传 | P0 | 支持 PDF、JPG、PNG |
| 文档合成 | 印章拖拽 | P0 | 鼠标拖拽调整位置 |
| 文档合成 | 印章变换 | P1 | 大小、旋转、透明度 |
| 导出 | PNG 下载 | P0 | 纯印章图片下载 |
| 导出 | PDF 导出 | P0 | 带印章的 PDF 文件 |

### 2.2 功能详情

#### 2.2.1 印章生成
- **公司名称输入**: 最长 20 个字符
- **样式选择**: 
  - 圆形：传统公章样式，双圆圈
  - 方形：财务章样式
  - 椭圆：椭圆形办公章
  - 五角星：特殊标识章
- **颜色自定义**: 
  - 边框颜色（默认红色 #d32f2f）
  - 文字颜色（默认红色 #d32f2f）
- **尺寸参数**:
  - 印章直径: 80-300px
  - 边框粗细: 1-10px

#### 2.2.2 文档合成
- **支持格式**: 
  - PDF (.pdf)
  - 图片 (.jpg, .jpeg, .png)
- **文件限制**: 最大 10MB
- **印章操作**:
  - 拖拽移动
  - 缩放: 30%-200%
  - 旋转: -180° ~ +180°
  - 透明度: 20%-100%

#### 2.2.3 导出功能
- **PNG 导出**: 透明背景/白色背景可选
- **PDF 导出**: 保留原文档尺寸，带印章层

## 3. 非功能需求

### 3.1 性能
- 页面加载时间 < 3秒
- 印章生成响应 < 500ms
- PDF 导出 < 5秒（10页以内）

### 3.2 兼容性
- Chrome 90+
- Edge 90+
- Firefox 90+
- Safari 14+
- 移动端适配（响应式布局）

### 3.3 用户体验
- 实时预览，所见即所得
- 操作提示文案清晰
- 错误提示友好

### 3.4 安全
- 文件本地处理，不上传服务器（客户端生成）
- 无需注册登录

## 5. 技术方案

### 5.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | 原生 HTML/CSS/JS | 轻量级，单页面应用 |
| 画布绘制 | Canvas API | 印章生成 |
| PDF 渲染 | PDF.js | 浏览器端 PDF 预览 |
| PDF 导出 | jsPDF | 客户端 PDF 生成 |
| 部署 | Vercel / Netlify | 免费 CDN 托管 |

### 5.2 架构
```
┌─────────────────────────────────────────┐
│                 浏览器                   │
├─────────────────────────────────────────┤
│  UI 层 (HTML + CSS)                     │
├─────────────────────────────────────────┤
│  业务逻辑层 (JavaScript)                │
│  ├── 印章生成模块                        │
│  ├── 文档处理模块                       │
│  └── 导出模块                           │
├─────────────────────────────────────────┤
│  第三方库                                │
│  ├── PDF.js (PDF渲染)                   │
│  └── jsPDF (PDF导出)                    │
└─────────────────────────────────────────┘
```
